# Seata-AT Demo

    seata-at文件夹下的Demo采用的是Seata的AT模式。
    
    Demo采用的是Spring-Boot + SpringAlibabaCloud + MyBatis + Seata 组件。

    运行Demo前，需要准备以下环境：
    1、MySQL
        将 ./seata/
    2、Nacos
        下载路径在 ./nacos/installer中。解压运行 bin目录下的文件即可。

    3、Seata
        下载路径在 ./seata/installer中。解压运行 bin目录下的文件即可。
